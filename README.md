# sysfox-1p-appstore

这是一个第三方 1Panel 应用商店仓库。

This is a third-party 1Panel App Store repository.

## 已包含的应用 / Included Apps

- **ChronoFrame** - 自托管照片展示和管理应用 / Self-hosted photo display and management application

## 使用方法 / Usage

在 1Panel 中添加此仓库作为第三方应用商店源：

Add this repository as a third-party app store source in 1Panel:

```
https://github.com/sysfox/sysfox-1p-appstore
```
