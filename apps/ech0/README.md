# Ech0

Ech0 是一款专为个人用户设计的新一代开源自托管平台，低成本、超轻量，支持 ActivityPub 协议，让你轻松发布和分享想法、文字与链接。简洁直观的界面结合高效的命令行工具，让内容管理变得简单而自由。你的数据完全自主可控，随时随地与世界联通，构建属于你的思想网络。

## 功能特性

### ☁️ 原子级轻量

- **内存占用不到 15MB** - 极低的资源消耗
- **镜像大小不到 40MB** - 快速部署
- **单 SQLite 文件存储** - 简单可靠的数据管理

### 🚀 极速部署

- **零配置** - 从安装到使用只需1条命令
- **无需额外依赖** - 开箱即用
- **跨平台支持** - Windows、Linux 以及树莓派等 ARM 架构设备

### ✍️ 零干扰写作

- **纯净的在线 Markdown 编辑器** - 专注于内容创作
- **丰富的 Markdown 插件** - 支持代码高亮、数学公式等
- **实时预览** - 所见即所得的编辑体验

### 📦 数据主权

- **所有内容存储于本地** - 数据完全由你掌控
- **支持 RSS 订阅** - 方便读者订阅你的更新
- **一键备份与恢复** - Web、TUI、CLI 三种模式支持
- **后台自动备份** - 数据安全无忧

### 🌍 ActivityPub 联邦

- **与 Mastodon、Misskey 等平台互联** - 融入联邦宇宙
- **去中心化生态** - 打破平台壁垒
- **跨平台互动** - 与其他联邦平台用户无缝交流

### 🎨 丰富功能

- **内置 Todo 管理** - 轻松管理待办事项
- **音乐播放器** - 沉浸式背景音乐与专注模式
- **视频分享** - 原生支持哔哩哔哩/YouTube 视频
- **快捷卡片** - 网站链接、GitHub 项目等富媒体内容展示
- **评论系统** - 支持 Twikoo 评论服务接入
- **S3 存储集成** - 原生适配 S3 兼容对象存储
- **OAuth2 集成** - 支持第三方登录与 API 授权

### 🔐 安全可靠

- **多账户权限管理** - 灵活的权限控制
- **访问令牌管理** - 一键生成与吊销
- **实时系统监控** - WebSocket 高性能监控面板
- **高可用 Webhook** - 事件驱动的自动化工作流

## 使用说明

部署后，访问配置的端口即可开始使用。首个注册的用户会自动被设置为管理员。

### 基本配置

1. **服务端口**：默认为 6277
2. **JWT 密钥**：建议修改为自定义的安全密钥
3. **数据目录**：所有数据存储在 `/app/data` 目录下
4. **备份目录**：备份文件存储在 `/app/backup` 目录下

### 进阶功能

- **ActivityPub 联邦**：将 Ech0 绑定域名后自动加入联邦宇宙
- **S3 存储**：在设置中配置 S3 兼容存储服务
- **评论系统**：在设置中填入 Twikoo 后端地址即可启用
- **Ech0 Connect**：实现多个 Ech0 实例间的互联与状态同步
